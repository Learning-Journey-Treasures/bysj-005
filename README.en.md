# 基于SSM的汽车租赁系统+毕业设计示例参考

#### Description
汽车租赁系统总共分为两个大的模块，分别是系统模块和业务模块。其中系统模块和业务模块底下又有其子模块。

业务模块包含：客户管理、车辆管理、出租单管理、检查单管理、统计分析等。

系统模块包含：菜单管理、角色管理、用户管理、数据源监控
